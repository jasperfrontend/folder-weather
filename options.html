🌤️ Folder Weather

Turn a bookmark-toolbar folder into a live weather readout.

Managed folders

Folders this extension is keeping up to date. You can also just name a toolbar folder [w Rotterdam] as a quick-add.

    Attach a folder

    e.g. {city} {temp}° makes the folder itself read “Rotterdam 24°”. Leave the global default blank to keep titles as just the city.

    Global settings

    Used by city-only folders (e.g. [w Dinteloord]) and as the editor default.

    Set e.g. {city} {temp}° to turn every managed folder's own name into a live readout. Applies to all folders that don't override it below.
